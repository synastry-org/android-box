Full Android Build Environment with Docker

* JDK 8;
* Android SDK 26;
* Android Build Tools 26.0.2;
* [Infer](http://fbinfer.com/) for static code analysis;
* [Redex](http://fbredex.com/) to optimize Android bytecode;
* Ruby.

### Build
`make`

### Run
`make run`
